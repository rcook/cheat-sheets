# IntelliJ keyboard shortcuts

| Action                 |                 |
|------------------------|-----------------|
| Debug                  | _Alt+Shift+F9_  |
| Toggle line breakpoint | _Ctrl+F8_       |
| Resume program         | _F9_            |
| Stop program           | _Ctrl+F2_       |
| Run program            | _Shift+F10_     |
| Recompile file         | _Ctrl+Shift+F9_ |
| Reformat code          | _Ctrl+Alt+L_    |
| Rename                 | _Shift+F6_      |
